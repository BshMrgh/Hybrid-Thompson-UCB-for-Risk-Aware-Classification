# Paper GitHub Page

This repository contains a static GitHub Pages paper website for:

**Risk-Sensitive Learning using Contextual Bandits for Medical Image Classification on Small Datasets**

Authors: Bashayer Marghalani and J. Michael Herrmann

## Files

- `index.html` — the paper webpage.
- `styles.css` — the visual styling.
- `assets/paper.pdf` — the manuscript PDF from the uploaded ZIP.
- `assets/*.png` — figures exported/optimised from the paper package.
- `.nojekyll` — tells GitHub Pages to serve the files as plain static files.
- `CITATION.cff` — draft citation metadata; update after acceptance or public release.

## How to publish on GitHub Pages

1. Create a new GitHub repository, for example `risk-sensitive-contextual-bandits-paper`.
2. Upload all files in this folder to the root of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder, then save.
6. Your page should appear at:
   `https://<your-github-username>.github.io/<repository-name>/`

## Important review-anonymity note

The current page includes author names and affiliations because they are present in the uploaded final LaTeX file. If the paper is still under double-blind review, do not publish this public version unless your venue explicitly allows it. For anonymous review, remove author names, emails, affiliations, acknowledgements, and any identifying links before making the page public.
